# Random Spectator Plugin
## Overview
t 秒に一回、n %の確率でプレイヤーがスペクテイターにされるプラグイン

## Version
Paper 1.15.2

## Usage
| cmd | arg[0] | arg[1] | description | default value |
| :---: | :---: | :---: | :--------- | :---:|
| randspec | help | | ヘルプ | |
| | start | | 起動 | |
| | stop | | 停止 | |
| | time | < long > | ゲームモード変更のインターバル（sec） | 30 |
| | ratio | < double > | ペクテイターになる確率（%） | 30 |
| | loadconfig | | コンフィグの読み出し（リロード）| |
